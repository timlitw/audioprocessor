# audioprocessor
cleanup raw listen to church streams

You had a singing service or program or wedding the soundman has other duties at church so he gets there turns the speaker system on and so the stream is started.

Now family or other want to listen and the file from listen to church has 30 - 60 minutes of either dead space or the sounds of people being ushered into place.
the folks that get the audio file do not know where the actaul event starts

This is an attempt to make a simple free audio processor that can do basic trim begining and end, trim dead spaces
find and take out background hum, ac noise, etc.

play thru to find and make small cuts 
maybe find the " I Do's" in a wedding and make them louder so they are audible
